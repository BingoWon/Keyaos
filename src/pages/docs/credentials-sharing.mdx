**Lower multiplier = higher priority = more usage = more earnings** (at a lower rate per request).

Subscription credentials (e.g., Google API with quota that refreshes periodically) default to `0.5` since the marginal cost to you is near zero. Credit-based credentials (e.g., OpenAI, Anthropic) default to `0.8` to reflect the real cost of your balance.

## Platform Fee

When your credential serves another user's request, **a 3% service fee** is deducted from your earnings. The consumer also pays a 3% fee on their side. **No user is ever charged more than 3% on any single API call.**

When you use your **own** credentials for your **own** requests (self-use), routing is **completely free** â€” no fees of any kind.

## Security

- Credentials are stored on Cloudflare's infrastructure with access limited to the routing engine
- Keyaos only forwards requests to pre-approved upstream endpoints â€” custom endpoints cannot be specified
- The platform monitors credential health and automatically disables credentials that return authentication errors
- Secrets are masked in all API responses and the dashboard UI
- Duplicate credentials are rejected to prevent conflicts
